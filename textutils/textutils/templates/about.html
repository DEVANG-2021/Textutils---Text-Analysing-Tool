About Textutils Tool

  • Textutil Tool is baiscally Text analysing tool.
  • This tool analyse your text and perform some of operations on it.
  • You can perform some of operation by using this tool, These operation includes :
    • You can Remove Punctuations from text by using this tool
    • You can Remove Spaces from text by using this tool
    • You can Remove Newlines from text by using this tool
    • You can Convert the text into Uppercase by using this tool
    • You can Count total numbers of the characters from text by using this tool
  • There is also one bug in this tool :
    • whenever you can try to perform character count operation and also any other operation it will only shows the result of Character count operation means it will only show total number of characters in the text and you can not perform any other operation with character count operation simulteniously.
